# System Design Study Repository

Este repositório reúne meus estudos sobre conceitos de system design, arquitetura de sistemas e boas práticas para construir aplicações escaláveis, resilientes e bem estruturadas.

O conteúdo está organizado em pastas por temas, como fundamentos, comunicação entre serviços, consistência, persistência, resiliência, observabilidade e outros tópicos relevantes.

⚠️ Ainda está em processo de construção, então o material pode ser atualizado ao longo do tempo conforme eu avanço nos estudos.

O objetivo aqui é servir como uma referência prática e pessoal para revisar conceitos importantes de forma organizada.
